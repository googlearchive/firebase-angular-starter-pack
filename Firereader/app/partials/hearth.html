A real-time RSS feed reader

Built with &

Firereader is a technical demonstration of how to use AngularJS together with Firebase. It is intended as a sample of how to use the two technologies together.

You can use it as your feed reader too, but it won't be as full featured as other alternatives. Currently you can create an account and manage a list of custom feeds. Feeds are updated in near real-time.

Fork Firereader on GitHub to build your own Angular + Firebase apps, or send pull requests to add functionality!
